# ReMarket — Buy & Sell Pre-Loved Treasures

AI-powered marketplace for pre-loved fashion, jewelry, watches, and more. Built for the UAE market.

## Live Demo
https://remarket-one.vercel.app

## Stack
- **Next.js 14** (App Router)
- **Framer Motion** (animations)
- **Tailwind CSS** (styling)
- **Anthropic Claude** (AI listing descriptions)
- **Vercel** (deployment)

## Quick Start
```bash
npm install
cp .env.example .env.local
npm run dev
```

## Deploy
```bash
npx vercel --prod
```

## Features
- Full landing page matching Lovable design
- Browse listings with filters (category, condition, search)
- Sell form with AI-generated descriptions
- Product cards with heart/like
- Responsive design
